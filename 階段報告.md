# 專案階段報告（截至 2026-03-28）

本文件摘要目前程式庫已落地之功能、資料庫變更與部署注意事項，供內部對齊與版本留存。

---

## 一、階段目標與範圍

以 **Next.js 課程站台 + 後台管理** 為主體，完成：

- 課程 CTA 雙模式（購物車／補助課）之前後台一致行為。
- 課程內容與側欄資訊之 **後台可編、前台依內容顯示**。
- 列表篩選與類別與 **資料庫 Category** 同步。
- Windows 本機開發體驗（終端機、環境檢查）之補強。

---

## 二、已完成項目（功能）

### 2.1 課程 CTA（`CourseCtaKind`）

- **購物車（CART）**：預設「加入購物車／立即購買」等文案與連結可於後台覆寫。
- **補助課（SUBSIDY）**：預設「預約諮詢／立即報名」語意；後台「價格、封面與本頁上架勾選」停用，儲存時不覆寫該等欄位（資料庫舊值保留）；前台不顯示定價／牌價／封面圖（改占位）、價格篩選與排序行為已區分補助課。
- 複製課程、匯出 MD、列表價格欄等已納入 CTA 類型之語意。

### 2.2 課程介紹分頁（前台）

- 「課程介紹／你可以學到／適合對象／學習前基本能力／學習前準備」：**後台未填則該區塊不顯示**；若整頁介紹區皆空，顯示簡短提示。
- 後台欄位與 Prisma 對應；並保留舊 Client 與 schema 不同步時之 raw 同步策略（learn／audience 相關）。

### 2.3 課程列表與搜尋

- 側邊／導覽類別與 **DB `Category`** 同步，連結使用 `?cat=<id>`（並保留舊有 tag 參數相容）。
- 補助課不納入「價格區間」篩選；「價格由低到高」排序時補助課排在後段。

### 2.4 前台側欄「課程資訊」

- 新增四則可選文字：**課程時長、單元結構、學習資源、完訓證明**。
- 後台於「基本資料」與「課程與單元 → 課程資訊」皆可編輯；**每欄有內容才顯示該列**；四欄皆空則不顯示整塊。
- **補助課**前台不顯示此整塊（文案仍存 DB，改回購物車課程後可生效）。

### 2.5 後台與工具

- 課程列表對補助課價格欄顯示「補助課（不適用）」。
- `dev.cmd`：便於在 Windows 以 cmd 啟動 dev（避開部分 PowerShell ExecutionPolicy 問題）。
- `.vscode/settings.json`：Windows 預設終端機改為 Command Prompt（可選共用）。
- 開發環境對 `DATABASE_URL` 之檢查已放寬（詳見 `lib/env-check.ts`）；`global-error` 文案已更新。
